# Фитнес-лендинг

Статический прототип трех экранов:

- первый экран с CTA;
- блок результатов с горизонтальной галереей и кнопками листания;
- форма заявки и FAQ-аккордеон.

## Локальный запуск

Откройте `index.html` в браузере. Сервер не нужен: форма, галерея и FAQ работают на обычном HTML/CSS/JS.

## GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите файлы из этой папки в корень репозитория.
3. Включите Pages: `Settings` -> `Pages` -> `Deploy from a branch`.
4. Выберите ветку `main` и папку `/root`.
5. После публикации адрес будет в формате `https://<username>.github.io/<repo>/`.

## GitLab Pages

1. Создайте новый проект на GitLab.
2. Загрузите файлы из этой папки в корень проекта.
3. GitLab CI/CD запустит `.gitlab-ci.yml`.
4. После успешного пайплайна адрес Pages появится в `Deploy` -> `Pages`.

Для реальных заявок на статическом хостинге нужно подключить внешний приемщик формы: Telegram-бот, Formspree, GetCourse, CRM или serverless-функцию.
